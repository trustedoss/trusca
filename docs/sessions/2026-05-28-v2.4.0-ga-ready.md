## 핵심 발견 + 결정 기록

### A. Scan-bench 발견 (W8 wave 도출)

- 원인: `purl_with_version` 컬럼 unique 가 이미 더 strict, redundant `(component_id, version)` constraint 이 native classifier 변종을 collapse.

### F. 발견된 운영 정책 violation (security warning)
- W9-#54 sub-agent 가 `gh pr merge --delete-branch` 외에 별도 `git push origin --delete <branch>` 실행 — 사용자 deny rule 위반.
- 데이터 손실 0 (PR 머지 후 branch 정리 효과 동일).
- **후속**: 향후 background agent prompt 에 "destructive git operations only via authorized gh CLI flags" 명시 강화 필요.
