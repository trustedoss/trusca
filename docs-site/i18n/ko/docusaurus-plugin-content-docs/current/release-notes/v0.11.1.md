# v0.11.1 — 테마 원복 + 새 로고

**UI / 브랜딩 패치 릴리스**입니다. 백엔드나 API 변경은 없습니다 — `0.11.0` 대비
프론트엔드 이미지, 문서, Helm 차트 메타데이터만 바뀝니다. 전체 머신 리더블
체인지로그는
[`CHANGELOG.md`](https://github.com/trustedoss/trusca/blob/main/CHANGELOG.md)에
있습니다.

## 주요 변경 사항

### 테마를 W11 라이트 테마로 원복

`0.11.0`의 W13 "Google AI Studio" 리스킨(흰 캔버스, 파란 primary, 알약형 버튼)을
**W11 Vercel + Linear** 스타일 — 오프화이트 캔버스, 웜 니어블랙 primary, 직각
모서리, 파란 Low 뱃지 — 로 되돌렸습니다. TRUSCA 브랜드와 개명은 영향이 없습니다.

### 새 로고 + 파비콘 세트

마크는 이제 **틸 체크 악센트가 있는 다크 슬레이트 타일**과 잉크색 "TRUSCA"
워드마크입니다. 전체 락업은 로그인 화면에 **"TrustedOSS SCA"** 태그라인을
더합니다. SVG와 함께 파비콘 세트가 제공됩니다: `favicon.ico`(16 / 32 / 48)와
iOS 홈 화면용 `apple-touch-icon.png`.

### 수정

- Helm 차트의 `icon:` URL이 존재하지 않는 경로를 가리키던 것을 게시된 마크
  래스터 이미지로 수정했습니다.

## 업그레이드 안내

UI 전용 변경입니다. `0.11.1` 이미지를 받거나 차트 `0.11.1`로 `helm upgrade`
하십시오. 마이그레이션이나 설정 변경은 필요 없습니다.
